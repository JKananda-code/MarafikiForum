# MarafikiForum.com - Next.js 15 + Supabase + Sanity + Stripe

Monetize ASAP stack for a developer skilled in Next.js 15, TypeScript, Tailwind.

## Stack
- Next.js 15 App Router
- Supabase (auth, db, realtime)
- Sanity.io (CMS for articles, sponsored posts)
- Stripe (memberships)
- TailwindCSS + shadcn/ui
- Vercel hosting
